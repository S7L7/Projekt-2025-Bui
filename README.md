# Projekt-2025-Bui

Jako projekt jsem si vybral tzv. "píchačku" k zaznamenání příchodu a odchodu zaměstnanců.

primární funkce bude zaznamenat příchod a odchod v reálném čase daného zaměstnance, spočítat dobu strávenou na pracovišti a poté časové údaje a údaje o zaměstnanci odeslat na server, kde by se data ukládali.
K tomuto projektu bude zapotřebí i hardware, rozhodl jsem se pro Raspberry Pi(S RTC modulem pro udržení reálného času i při výpadku elektřiny), RFID čtečku karet a obyčejné čipy s frekvencí 13.67MHz.

Kod budu tvořit za pomocí C++ a SQL Database. Jak to dopadne? To se dozvíme na konci pořadu.
